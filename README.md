# AtomLearn AI Dashboard

AtomLearn AI Dashboard is an AI-powered Adaptive Learning Management System designed to provide students with a smart, personalized, and interactive learning experience. The system combines modern UI/UX design with student performance analytics to create an intelligent educational platform.

The project includes a futuristic login interface with animated splash screen effects, secure login functionality, a personalized student dashboard, course performance tracking, and visual analytics using dynamic charts.

## Features

* AI-powered Adaptive LMS Interface
* Modern Glassmorphism UI Design
* Animated Splash Screen with Rotating AI Robot
* Secure Student Login System
* Personalized Dashboard
* Student Learning DNA Section
* Course Progress Monitoring
* Interactive Course Detail View
* Dynamic Analytics Dashboard
* Performance Charts using Chart.js
* Responsive and Attractive Design
* Flask + HTML + Tailwind CSS Integration
* MySQL Database Connectivity
